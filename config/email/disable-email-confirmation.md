# Désactiver la confirmation d'email temporairement

## 🔧 Étapes dans Supabase Dashboard :

1. **Allez sur** : https://supabase.com/dashboard/project/kvwzbofifqqytyfertkh/auth/users
2. **Cliquez sur "Settings"** (dans le menu Authentication)
3. **Trouvez "Enable email confirmations"** 
4. **Décochez cette option**
5. **Cliquez "Save"**

## ✅ Résultat :
- Les nouveaux utilisateurs pourront se connecter immédiatement
- Pas besoin d'email de confirmation
- Parfait pour le développement et les tests

## 🔄 Pour réactiver plus tard :
- Recochez "Enable email confirmations" 
- Configurez un service SMTP transactionnel (SendGrid, Mailgun, etc.)

## 🚀 Test immédiat :
Une fois désactivé, vous pourrez :
1. Créer un nouveau compte
2. Vous connecter immédiatement
3. Voir l'application complète !

## 💡 **PROBLÈMES RÉSOLUS PAR CET AGENT**

### **1. Build Failed - Native Module**
```
Error: Unable to resolve module @react-native-community/...

→ Solution détaillée dans agent avec :
  - Installation correcte
  - Clear caches
  - Configuration eas.json
  - Re-build avec cache clear
```

### **2. Build Failed - Gradle**
```
Error: Execution failed for task ':app:mergeReleaseResources'

→ Solution détaillée avec :
  - Vérification build.gradle
  - Clean Gradle
  - Config minSdk/targetSdk
  - Re-build
```

### **3. iOS Signing Error**
```
Error: No signing certificate "iOS Distribution" found

→ Solution avec :
  - Vérification certificats
  - Génération via EAS
  - Re-build
```

### **4. APK/IPA Too Large**
```
Warning: APK size > 100MB

→ Optimisations :
  - Activer Hermes
  - Compresser images
  - Tree shaking
  - Bundle analysis
```

### **5. Google Play Rejection**
```
Rejection: "Your app crashed during testing"

→ Plan d'action :
  - Analyser Pre-Launch Report
  - Reproduire crash
  - Fix + nouvelle version
  - Re-soumission
```

### **6. App Store Rejection**
```
Rejection: "Guideline 2.1 - Performance"

→ Plan d'action :
  - Lire message complet
  - Fournir compte démo
  - Notes pour reviewers
  - Fix + re-soumission
```

## 🛠️ **COMMANDES PRINCIPALES**

### **Build Production**
```bash
# Android AAB (pour Google Play)
eas build --platform android --profile production

# iOS IPA (pour App Store)
eas build --platform ios --profile production

# Both platforms
eas build --platform all --profile production
```

### **Submit to Stores**
```bash
# Submit to Google Play
eas submit --platform android --profile production

# Submit to App Store
eas submit --platform ios --profile production
```

### **Troubleshooting**
```bash
# Clear cache et rebuild
eas build --platform android --clear-cache

# Check build status
eas build:list

# Download build
eas build:download --id [BUILD_ID]
```

## 📝 **Conventions de Nommage**

- **MAJUSCULES** : Tous les fichiers documentation
- **Underscores** : Séparateurs de mots (ex: `FORM_MIGRATION_GUIDE.md`)
- **Préfixes** :
  - `AUDIT_` : Audits et analyses
  - `FIX_` : Corrections de bugs
  - `GUIDE_` : Guides détaillés
  - `QUICK_` : Références rapides
  - `FINAL_` : Versions finales
  - `SYSTEM_` : Documentation systèmes

## 🔄 **Maintenance**

### **Ajouter un Nouveau Document**
1. Identifier la thématique (testing, chat, design, etc.)
2. Placer dans le dossier approprié
3. Utiliser les conventions de nommage
4. Mettre à jour cet INDEX.md si nécessaire

### **Archiver un Document**
1. Déplacer vers `archive/`
2. Ajouter note "OBSOLETE" en début de fichier
3. Référencer le nouveau document si remplacement

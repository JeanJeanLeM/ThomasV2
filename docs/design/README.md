## 🎨 Composants Design System

### **Atoms (Atomes)**
- Colors, Typography, Spacing, Shadows
- Buttons, Inputs, Icons

### **Molecules (Molécules)**
- FormField, SearchBar, Card
- DatePicker, Dropdown, Modal

### **Organisms (Organismes)**
- Header, BottomNav, Forms
- Lists, Grids, Sections

### **Templates**
- FormScreen, ListScreen, DetailScreen
- ModalScreen, EmptyState

## 📐 Principes

- **Mobile-First** - Conçu d'abord pour mobile
- **Responsive** - S'adapte à toutes les tailles
- **Accessible** - WCAG 2.1 AA
- **Cohérent** - Design system unifié
- **Performant** - Optimisé pour React Native

## 🔗 Implémentation

**Dossier** : `src/design-system/`
- `atoms/` - Composants de base
- `molecules/` - Composants composés
- `organisms/` - Composants complexes
- `templates/` - Templates écrans
- `theme/` - Tokens design (colors, spacing, etc.)

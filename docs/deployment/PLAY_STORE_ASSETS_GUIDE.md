# 🎨 Guide Assets Google Play Store

## 📋 Assets Requis

### 1. Icon App (512x512) ✅
**Fichier** : `assets/ThomasSmall.png`

**Vérification** :
- [ ] Dimensions : 512x512 pixels exactement
- [ ] Format : PNG 32-bit avec transparence
- [ ] Qualité : Net et lisible

**Si besoin de redimensionner** :
- Utiliser un outil comme GIMP, Photoshop, ou Canva
- Maintenir les proportions
- Exporter en PNG 32-bit

---

### 2. Feature Graphic (1024x500) ⚠️ À CRÉER

**Spécifications** :
- Taille : 1024x500 pixels exactement
- Format : PNG ou JPG
- Poids max : 1MB

**Contenu suggéré** :
```
[Logo Thomas centré]
Thomas V2
Assistant IA pour Agriculteurs

[Éléments visuels : icônes agriculture, chat IA]
[Couleur de fond : #22c55e ou dégradé vert]
```

**Outils recommandés** :
- **Canva** : Template "Google Play Feature Graphic" (gratuit)
- **Figma** : Design professionnel
- **Photoshop** : Design avancé
- **GIMP** : Gratuit et puissant

**Template Canva** :
1. Aller sur canva.com
2. Rechercher "Google Play Feature Graphic"
3. Utiliser le template 1024x500
4. Personnaliser avec logo Thomas
5. Télécharger en PNG haute qualité

---

### 3. Screenshots (Minimum 2, Idéal 4-8) ⚠️ À CRÉER

**Spécifications** :
- Taille recommandée : 1080x1920 (portrait) ou 1920x1080 (landscape)
- Format : PNG ou JPG
- Poids max : 8MB par image
- Minimum : 2 screenshots
- Idéal : 4-8 screenshots

**Screenshots à prendre** :

#### 1. Écran Chat avec Thomas Agent 🤖
- Montrer une conversation naturelle
- Exemple : "J'ai observé des pucerons sur mes tomates"
- Réponse de Thomas avec suggestions
- Interface chat claire et lisible

#### 2. Dashboard / Statistiques 📊
- Vue d'ensemble de l'exploitation
- Graphiques et métriques visibles
- Tâches en cours
- Design moderne

#### 3. Liste des Tâches ✅
- Tâches terminées et planifiées
- Avec photos et descriptions
- Filtres et recherche visibles
- Interface intuitive

#### 4. Observations Terrain 👁️
- Observation avec photo
- Catégorisation (maladie, ravageur, météo)
- Localisation parcelle
- Informations détaillées

#### 5. Gestion Parcelles/Matériels 🗺️
- Liste des parcelles
- Informations détaillées
- Carte si disponible
- Organisation claire

#### 6. Profil / Paramètres ⚙️
- Gestion ferme
- Multi-exploitations
- Paramètres utilisateur
- Interface épurée

**Comment créer les screenshots** :

**Méthode 1 : Émulateur Android**
1. Lancer Android Studio Emulator
2. Installer l'APK sur l'émulateur
3. Naviguer vers chaque écran
4. Prendre capture (Ctrl+S ou bouton caméra)
5. Screenshots sauvegardés automatiquement

**Méthode 2 : Device Physique**
1. Installer l'APK sur votre téléphone
2. Naviguer vers chaque écran
3. Prendre screenshot (Power + Volume Down)
4. Transférer sur PC
5. Redimensionner si nécessaire (1080x1920)

**Améliorer les screenshots** :
- Ajouter un cadre de téléphone (mockup)
- Utiliser Screely.com ou MockUPhone.com
- Ajouter des annotations/légendes (optionnel)
- Utiliser Figma avec templates

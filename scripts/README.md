# Scripts du projet Thomas V2

Ce dossier regroupe les scripts utilitaires (Node.js, SQL, shell, PowerShell).

## Scripts PowerShell (`.ps1`)

- **debug_android.ps1**, **debug-thomas-*.ps1** – Lancement / debug de l’app (Android, clavier, minimal, etc.)
- **rebuild_android_audio_fix.ps1** – Rebuild Android avec correctifs audio
- **SEARCH_KEYSTORE.ps1** – Recherche du keystore de signature

Exécution depuis la racine du projet :  
`.\scripts\NomDuScript.ps1`

## Scripts Node.js (`.js`)

Tests, validation, import de données, diagnostic Supabase/prompts, etc.

## Scripts SQL (`.sql`)

Migrations ponctuelles, correctifs prompts, diagnostic (à exécuter dans Supabase ou en local).
